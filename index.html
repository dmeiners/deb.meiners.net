About

This is a home for packaging NGINX deb packages for Ubuntu.

I build and maintain clean, up-to-date Debian packages of the latest stable and legacy NGINX releases for the current Ubuntu LTS versions. No bloat, no extra patches — just reliable packages built directly from official sources.
